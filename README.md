# 🚚 TransportCompanyApp (Система управления грузоперевозками)

Полноценное десктопное приложение для автоматизации работы транспортной компании. Разработано на C# WinForms.

## 📌 О проекте

Система предназначена для учёта заказов на грузоперевозки: клиенты, водители, транспорт, статусы заказов. Реализовано создание новых заказов, завершение существующих и формирование отчётов с аналитикой.

---

## ✨ Функциональные возможности

### 🔐 Авторизация
- Вход по логину и паролю

### 📦 Управление заказами
- Таблица заказов (номер, клиент, водитель, транспорт, статус, откуда/куда, стоимость, дата)
- Поиск по заказам
- Фильтрация и обновление списка

### ➕ Создание нового заказа
- Выбор клиента, водителя, транспорта
- Ввод адресов (откуда/куда)
- Указание стоимости заказа
- Автоматическое присвоение статуса "New"

### ✅ Завершение заказа
- Изменение статуса на "Delivered"

### 📊 Отчёт по деятельности
- Общая статистика: всего заказов, доставлено, в пути, в обработке, новых
- Общий доход и средняя стоимость заказа
- **ТОП-5 клиентов** (по количеству заказов и сумме)
